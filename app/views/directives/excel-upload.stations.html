Understanding the Excel Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Station Code – Required. Alphanumeric and 20 characters maximum allowed.
  • Effective From – Required. Effective Date of the Station, Format = yyyymmdd.
  • Effective To – Required. End Date of the Station, Format = yyyymmdd.
  • Catering Station – Required. Boolean value. Format = yes or no.
