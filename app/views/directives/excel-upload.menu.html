Understanding the Excel Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Menu Code – Optional. Alphanumeric and 20 characters maximum allowed. If missing replace by Menu Name.
  • Menu Name – Required. Alphanumeric and 25 characters maximum allowed.
  • Menu Start Date – Required. Effective Date of the Menu, Format = yyyymmdd.
  • Menu End Date – Required. End Date of the Menu, Format = yyyymmdd.
  • Item Code – Required. Item that is part of the Menu; should match Item Code already setup.
  • Item Name – Required. Item that is part of the Menu; should match Item Name already setup.
    Note: Item Code should match Item Name already setup.
  • Qty – Required. Item quantity for the Menu; Numeric.Number
