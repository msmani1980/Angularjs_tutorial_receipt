Understanding the Excel Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Employee ID - Required. This is the employee's ID and supports up to 15 characters.
  • Passcode - Optional. This is for any password used by the employee in an external system that can be later connected to TS.
  • First Name - Required. This is the employee first name and supports up to 25 characters.
  • Last Name - Required. This is the employee last name and supports up to 25 characters.
  • Country - Required. This should match the countries that the company operates e.g. United Kingdom.
  • Base Station - Required. This is the base station of employee and should be related to the Country above, and also match stations that the company operates. Only one allowed per row and all capital.
  • Employee Title - Required. What is the employee's job title, e.g. Crew Manager, and supports up to 25 characters.
  • Effective Start Date - Required. Effective Date of the employee, Format = yyyymmdd. Current and future dates allowed.
  • Effective End Date - Required. End Date of the employee, Format = yyyymmdd. Current and future dates allowed.
