Understanding the Excel Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Catering Station – Required. This should match the 'LMP Station' field.
  • Item Code – Required. This is the Retail Item Code of the item and should match with the item name.
  • Item Name – Required. This is the Retail Item Name of the item and should match with the item code.
  • Quantity – Required with 0 being a valid count. This should be numeric value without decimals.
