Understanding the Excel Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Operating Currency Code - Required. This is the Operating Currency Code of retail company and supports up to 3 characters.
  • Accepted Currency Code - Required. This is the Accepted Currency Code of retail company and supports up to 3 characters.
  • Start Date - Required. Start Date of the Report Exchange Rate, Format = yyyymmdd. Current and future dates allowed.
  • End Date - Required. End Date of the Report Exchange Rate, Format = yyyymmdd. Current and future dates allowed.
  • Exchange Rate - Required. This is the exchange rate value of operating and accepted currency codes. It is a numeric value and supports up to 4 decimal values.
