Download Template

The file is an Excel (.xls, .xlsx) file and has only one sheet.
The first line should be the header with the column names.

The column order and name should match the following:

  • Number – Required. This is the Flight Number of the Flight Schedule.
  • Aircraft Type – Optional. Should match an Aircraft Type setup in the system.
  • Tail Number – Required if Aircraft Type is entered. Should match a Tail Number setup in the system.
  • Seat Config – Required if both Aircraft Type and Tail Number is entered. Should match a Seat Config setup in the system.
  • Departure Station – Required. This is the 3-letter airport code.
  • Departure Time – Required. This is the time of Flight Departure (HH:MM). Enter in 24 hour format. (Ex. 15:24 for 3:24PM or 10:15 for 10:15AM)
  • Arrival Station – Required. This is the 3-letter airport code.
  • Arrival Time – Required. This is the time of Flight Arrival (HH:MM). Enter in 24 hour format. (Ex. 15:24 for 3:24PM or 10:15 for 10:15AM)
  • Days of Operation – Required. Number representing the day of the week. 1 = Monday, 2 = Tuesday, 3 = Wednesday, 4 = Thursday, 5 = Friday, 6 = Saturday, 7 = Sunday. Multiple days can be comma separated.
  • Block Time – Optional. 6 digit whole number e.g. 123456 (not 123456.00).
  • Ground Time – Optional. 6 digit whole number e.g. 123456 (not 123456.00).
  • Trip Distance – Optional. 5 digit whole number e.g. 12345 (not 12345.00).
  • Trip Distance - Miles (mi) or Kilometer (km) – Optional. mi to represent miles or km to represent kilometer.
  • Previous Number – Optional. This is the previous Flight Number of this Flight Schedule.
  • Next Number – Optional. This is the next Flight Number of this Flight Schedule.
  • First Trip of Day – Optional. Y for Yes and N for No to represent if this is the first flight for the day.
  • Last Trip of Day – Optional. Y for Yes and N for No to represent if this is the last flight for the day.
  • Schedule Effective Date – Effective Date of the Flight Schedule, Format = yyyymmdd
  • Schedule End Date – End Date of the Flight Schedule. Format = yyyymmdd
